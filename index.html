What's New

Vector 2.0 —
Everything Changed.

The biggest update since Vector launched. 2.0 isn't a collection of new features bolted onto the old system — it's a rebuilt foundation that makes everything more reliable, more capable, and more consistent.

```
FOUNDATION
🔧

Tool Calling Architecture

The most important change in 2.0 isn't a feature you see — it's the engine underneath everything. Previous versions relied on complex prompt engineering to guide the AI's behavior. That approach was inherently fragile: models could misinterpret instructions, hallucinate steps, or produce unpredictable results under pressure.

Vector 2.0 replaces all of that with a proper structured tool-calling system. The AI selects from a clearly defined set of tools, issues machine-readable commands, sees the results, and continues the loop — with no guessing and no improvisation.

Dramatically more consistent behavior, cleaner multi-step execution, far fewer failures, and a foundation that scales as new tools and models are added.

👁

Vision — Show Vector Anything

Ask Vector to look at an image and it opens an image picker. Hand it a photo, document, receipt, or screenshot — it passes the image to the AI, which reads and interprets its contents before responding. Identify objects, extract text, or ask what's in any picture.

💬

Follow-Up Questions

After every response, Vector asks "Anything else?" — keeping the conversation alive without relaunching. Full history is preserved throughout the session so follow-ups are answered with complete context every time.

⚡

Two Powerful New Models

Claude Haiku from Anthropic — fast, capable, and strong across reasoning and tool use. Gemini 2.5 Flash Lite from Google — impressive everyday performance at exceptional efficiency. Both join a growing lineup with more models always on the way.

```
