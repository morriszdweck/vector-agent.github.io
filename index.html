What's New

Vector 2.0 —
Everything Changed.

The biggest update since Vector launched. 2.0 isn't features bolted onto the old system — it's a rebuilt foundation that makes everything more reliable, capable, and consistent.

FOUNDATION
🔧

Tool Calling Architecture

The most important change in 2.0 isn't visible — it's the engine underneath everything. Previous versions relied on complex prompt engineering to guide behavior. That was fragile: models could misinterpret instructions, hallucinate steps, or produce unpredictable results under pressure.

Vector 2.0 replaces all of that with a proper structured tool-calling system. The AI selects from a defined set of tools, issues machine-readable commands with specific arguments, sees the results, and continues the loop — no guessing, no improvisation.

Dramatically more consistent behavior, cleaner multi-step execution, far fewer failures, and a foundation built to scale.

👁

Vision — Show Vector Anything

Ask Vector to look at an image and it opens an image picker. Hand it a photo, document, receipt, or screenshot — it passes the image to the AI which reads and interprets the contents before responding. Identify objects, extract text, analyze documents.

💬

Follow-Up Questions

After every response, Vector asks "Anything else?" — keeping the conversation alive without relaunching. Full history is preserved so every follow-up is answered with complete context from the entire session.

⚡

Two Powerful New Models

Claude Haiku from Anthropic — fast, capable, strong reasoning. Gemini 2.5 Flash Lite from Google — impressive everyday performance at exceptional efficiency. Both join a growing lineup with more models always coming.
